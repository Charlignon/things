## Congigurer Blue Marble
Blue Marble est un outil qui permet de superposer un modèle directement sur la grille de wplace. Voilà comment le configurer
1. Installer [ViolentMonkey](https://violentmonkey.github.io/) ou [TamperMonkey](https://www.tampermonkey.net/)
   - Ces outils permettent d'exécuter des scripts dans le navigateur ⚠️ Il faut être très prudent et ne pas installer n'importe quoi ! Des scripts malveillants pourraient voler toutes vos données et même faire des actions à ta place. Celui recommandé ci-desous est sûr.
2. Installer le script [Blue Marble](https://github.com/SwingTheVine/Wplace-BlueMarble?tab=readme-ov-file#installation-instructions)
4. Rafraîchir la page de wplace
5. Se déplacer jusqu'à l'endroit indiqué dans chaque projet, et cliquer sur le pixel d'origine. Il est aussi possible de rentrer les coordonées manuellement directement dans la popup Blue Marble
6. Uploader le sprite
7. Clicker sur Create
8. C'est prêt ! Plus qu'à dessiner :)
